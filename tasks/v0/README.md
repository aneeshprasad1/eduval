# EduVal v0 Tasks

50 pilot tasks across Math (25) and Physics (25).

## Math (25 tasks)
- **Algebra** (8): linear equations, quadratics, systems, inequalities, exponents, functions
- **Geometry** (8): triangles, area/perimeter, circles, similar triangles, coordinate geometry, volume, transformations  
- **Calculus** (9): limits, derivatives (power rule, chain rule, product rule), integrals, optimization

## Physics (25 tasks)
- **Mechanics** (13): Newton's laws, kinematics, energy, momentum, friction, inclined planes, projectile motion, springs, circular motion, torque
- **Electromagnetism** (12): Coulomb's law, electric field, Ohm's law, series/parallel circuits, potential, magnetic force, capacitance, induction, Gauss's law, RC circuits

## Task Schema
Each task includes: `id`, `subject`, `topic`, `subtopic`, `difficulty` (1-4), `problem`, `correct_answer`, `solution_steps`, `common_misconceptions`, `scaffolding_hints`, `prerequisite_knowledge`.
